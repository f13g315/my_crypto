# my_crypto
Python program to perform select cryptographic functions, such as symmetric encryption/decryption, asymmetric encryption/decryption, and key pair generation.

# Installation and Running Instructions
1. Download and install latest version of python
2. Download my_crypto.py and place into directory of your choice
3. Open command prompt and navigate to directory where you placed my_crypto.py
4. Use command "python3 my_crypto.py" to run the program
